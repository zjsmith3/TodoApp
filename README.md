# App Name

A simple Android application created for my Mobile Application Development course.  
The app demonstrates basic Android concepts such as ViewModels, Room database usage, state handling, and Jetpack Compose UI.

## Features
- Add a task/item
- Delete a task/item
- Mark tasks as completed (My added feature)
- Edit existing tasks (my second added feature)
- Persistent storage using Room Database
- Clean UI built with Jetpack Compose

## Technologies Used
- Kotlin
- Android Studio (Electric Eel/Latest)
- Jetpack Compose
- Room Database
- MVVM Architecture
- ViewModel & LiveData/StateFlow

## How to Run
1. Clone the repo:  
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git`
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or physical device
